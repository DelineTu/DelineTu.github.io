三、创新性描述

1.本程序全部内容由本人独立完成。

2.本程序的第一大难点在于公式较高的复杂度。为解决此问题，提高编程及运行效率，并且方便呈现效果的调整，我将公式部分参数作为外部变量取定，尽量通过调整ax.quiver的参数值而非公式本身来调整图像；另外，对于在公式中多次出现的三角函数值，因其较难通过坐标表示，故我将三角函数值的计算放在公式之前，并应用numpy库中的数学函数实现了函数值的计算。

3.本程序的第二大难点在于电荷附近干扰图像的去除。因电荷附近电场强度较大，导致影响其他部分可视化效果，故需要隐去电荷附近部分图像，而仅在linspace函数中去除中间点，将导致三维空间除中间部分之外的部分也被成块去除，为解决此问题，我研究了np.meshgrid函数生成点矩阵的原理，对于meshgrid函数的返回值进行修改，实现了部分空间图像的去除。该过程花费时间超过两小时。

4.本程序的第三大难点在于可视化范围的调整和参数的取定。为得到更加美观合理的可视化图像，我在选取展示范围，确定参数上进行了较多尝试，具体过程见第四部分成果展示中的说明。

5.本程序的第四大难点在于计算驻点过程中，偏导数公式的简化。因所生成偏导数浮点型系数数目较多，导致解方程组运行缓慢，我将偏导数系数全部进行四舍五入处理，提高了程序运行效率。

import numpy as np
import matplotlib.pyplot as plt
from matplotlib.patches import Circle

nq = 2 # 电荷数目
charges = []
for i in range(nq):
    if i % 2 == 0:
        q = 1
    else:
        q = 1
    charges.append((q, (np.cos(2 * np.pi * i / nq), np.sin(2 * np.pi * i / nq))))
charges.append((-2, (0, 0)))

fig, ax = plt.subplots(figsize=(4, 4), dpi= 300)

charge_colors = {True: '#aa0000', False: '#0000aa'}
for q, pos in charges:
    ax.add_artist(Circle(pos, 0.05, color=charge_colors[q > 0]))

plt.text(-0.2, -0.3, '-2q', fontsize=10, color='black')
plt.text(-1.2, -0.3, '+q', fontsize=10, color='black')
plt.text(+0.8, -0.3, '+q', fontsize=10, color='black')
plt.grid()
ax.set_xlabel('$x$')
ax.set_ylabel('$y$')
ax.set_xlim(-2, 2)
ax.set_ylim(-2, 2)
ax.set_aspect('equal')
plt.show()

from matplotlib.patches import Circle

def E( q, r0, x, y):
    den = np.hypot(x - r0[0], y - r0[1]) ** 3
    return q * (x - r0[0]) / den, q * (y - r0[1]) / den

# 对x,y进行网格化取点
nx, ny = 64, 64
x = np.linspace(-200, 200, nx)
y = np.linspace(-200, 200, ny)
X, Y = np.meshgrid(x, y)

l = 2
nq = l
#电荷数目
charges = []
for i in range(nq):
    if i % 2 == 0:
        q = 1
    else:
        q = 1
    charges.append((q, (np.cos(2 * np.pi * i / nq), np.sin(2 * np.pi * i / nq))))
charges.append((-l, (0, 0)))

# 电场矢量，e=（ex，ey），作为独立分量
Ex, Ey = np.zeros((ny, nx)), np.zeros((ny, nx))
for charge in charges:
    ex, ey = E(*charge, x=X, y=Y)
    Ex += ex
    Ey += ey
fig, ax = plt.subplots(figsize=(4, 4), dpi= 300)




color = 2 * np.log(np.hypot(Ex, Ey))
ax.streamplot(x, y, Ex, Ey, color=color, linewidth=1, cmap=plt.cm.magma,
              density=2, arrowstyle='fancy', arrowsize=0.5)

#为电荷添加实心圆
charge_colors = {True: '#aa0000', False: '#0000aa'}
for q, pos in charges:
    ax.add_artist(Circle(pos, 0.05, color=charge_colors[q > 0]))

ax.set_xlabel('$x$')
ax.set_ylabel('$y$')

ax.set_aspect('equal')
plt.show()
